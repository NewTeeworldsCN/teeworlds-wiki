# DM 与 TDM
DM 与 TDM 是 Teeworlds中内置的原版模式之一。

## 玩法
在DM模式中，玩家各自为战，靠击杀他人得分。

在TDM模式中，玩家被分为红蓝两队，靠击杀敌方队伍得分，击杀己方队伍玩家则扣分。

当队伍或个人分数达到目标时，游戏将会结算。在TDM中，如果倒计时结束但两队分数相等，将进入加时赛。
## 变体

DM模式存在诸多修改变体，

以及一些和其他模式结合的另类变体。

### 战地
[战地](?id=modes/battlefield)，即**Battlefield**，虽然主要是CTF模式的特殊变体，但是可以切换到TDM模式。

### 瞬杀
瞬杀，即**instagib**，是一种特殊的PVP玩法。

在该玩法下，武器造成的伤害将造成秒杀的效果。

瞬杀类夺旗的代表为 **gDM** 和 **iDM** 以及它们的TDM版本。

### Hunter
DM模式的变体，玩家虽分为两个阵营，平民和猎人，但是不清楚队友，

需要通过识别队友，击杀不属于自己阵营的玩家。

猎人的武器造成的伤害将是平民的2倍，并且榴弹附带破片效果，且不会对自己造成伤害。

猎人自带一柄锤子，而平民没有。

目前活跃的分支为Hu1night的DDNet-PVP-HunterN
