# LMS 与 LTS
LMS 与 LTS 是 Teeworlds0.7开始内置的原版模式之一，全称分别为**Last Man Standing**与**Last Team Standing**

## 玩法
在这类生存模式下，所有玩家每回合都只有一条命。

在LMS模式中，玩家各自为战，靠击杀他人得分。

在LTS模式中，玩家被分为红蓝两队，靠击杀敌方队伍得分，击杀己方队伍玩家则扣分。

在一局LMS/LTS游戏中，会有多个回合。

当经过多个回合后，队伍或个人分数达到目标时，游戏将会结算。

在LMS中，如果此时有两名及以上最高分玩家，将进入加时赛，

在LTS中，如果此时两队分数相等，将直接平局。

## 变体

LMS模式存在较少的变体。

### LMB
Last Man Blocking 是 LMS 模式的变体，

该模式的击杀方式变成了Block模式的将人丢进水(Freeze)中。
